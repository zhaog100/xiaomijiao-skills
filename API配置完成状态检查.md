## 🔑 API配置方案

### 方案1: OpenAI API (推荐)
**配置步骤:**
1. 访问: https://platform.openai.com/
2. 注册账号并获取API密钥
3. 设置环境变量:
   ```powershell
   $env:OPENAI_API_KEY = "sk-your-openai-key"
   ```
4. 验证配置:
   ```bash
   & "C:\Users\zhaog\AppData\Roaming\npm\pi.ps1" --list-models
   ```
