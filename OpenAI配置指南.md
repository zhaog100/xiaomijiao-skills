### 配置步骤

#### 1. 访问OpenAI官网
```
网址: https://platform.openai.com/
```
- 点击 "Sign up" 注册账号
- 或使用Google账号快速注册

#### 2. 获取API密钥
```
步骤:
1. 登录后点击 "API keys"
2. 点击 "Create new secret key"
3. 输入密钥名称 (如: QMD-Knowledge-Base)
4. 点击 "Create secret key"
5. 复制生成的密钥 (格式: sk-xxxxxxxxxxxxxxxxxxxxx)
```

#### 3. 配置环境变量
在PowerShell中执行：

```powershell
# 设置OpenAI API密钥
$env:OPENAI_API_KEY = "sk-your-openai-key-here"

# 可选: 永久保存到用户环境变量
[Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-your-openai-key-here", "User")
```

#### 4. 验证配置
```bash

```

### 注意事项

#### 1. 密钥安全
- 不要在代码中硬编码API密钥
- 不要与他人分享API密钥
- 定期轮换API密钥

#### 2. 网络要求
- 需要稳定的网络连接
- 建议使用VPN或代理访问
- 检查防火墙设置

#### 3. 使用限制
- API有速率限制
- 新用户有使用限制
- 需要绑定支付方式 (即使使用免费额度)

### 常见问题解决

#### 问题1: API密钥无效
```
解决方法:
1. 检查密钥格式是否正确 (以sk-开头)
2. 确认账号已验证邮箱
3. 检查密钥是否已过期
```

#### 问题2: 余额不足
```
解决方法:
1. 绑定支付方式
2. 充值获得更多额度
3. 使用更便宜的模型
```

#### 问题3: 速率限制
```
解决方法:
1. 降低请求频率
2. 使用不同的模型
3. 等待限制重置
```
