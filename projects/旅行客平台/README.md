## 🎯 目录结构说明

### 第一层：测试用例 / 自动化测试

**测试用例**：CSV格式的测试用例文档（手工测试）

**自动化测试**：Playwright脚本（自动化执行）

### 第二层：系统模块

**系统管理** > **租户管理** > **8个功能模块**

按照系统菜单结构组织，而不是按测试策略

### 第三层：测试策略

**功能测试**：正常功能测试

**异常测试**：
- 输入验证测试
- 安全性测试
- 容错性测试
- 性能测试
