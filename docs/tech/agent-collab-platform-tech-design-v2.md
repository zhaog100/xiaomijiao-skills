## 🏗️ 1. 整体架构设计

### 1.1 双技能包架构

```
┌─────────────────────────────────────────┐
│         飞书开放平台                     │

              ↓

└─────────────┬───────────────────────────┘
              │              │
              ↓              ↓
    ┌──────────────┐  ┌──────────────┐
    │ agent-a-pm   │  │ agent-b-dev  │
    │ -skill       │  │ -skill       │

    └──────────────┘  └──────────────┘
              │              │
              ↓              ↓
        共享数据库（SQLite）
```

## 📊 2. 状态定义（4阶段26状态）

### 2.1 产品构思阶段（6状态）

| 状态 | 英文 | 说明 | 触发条件 |
|------|------|------|---------|
| 草稿 | draft | 初始创建 | 新建产品 |
| 待审核 | pending_review | 提交审核 | PM完成PRD |
| 已批准 | approved | 审核通过 | 官家批准 |
| 已拒绝 | rejected | 审核拒绝 | 官家拒绝 |
| 已暂停 | paused | 暂停开发 | 优先级调整 |
| 已取消 | cancelled | 取消项目 | 不再开发 |

**状态流转**：
```
draft → pending_review → approved/rejected
                     ↓
                   paused
                     ↓
                 cancelled
```

---

### 2.2 技术设计阶段（6状态）

| 状态 | 英文 | 说明 | 触发条件 |
|------|------|------|---------|
| 设计中 | designing | 开始设计 | 产品approved |
| 待审核 | pending_review | 设计完成 | Dev完成设计 |
| 已批准 | approved | 审核通过 | PM批准 |
| 已拒绝 | rejected | 审核拒绝 | PM拒绝 |
| 已暂停 | paused | 暂停设计 | 技术风险 |
| 已取消 | cancelled | 取消设计 | 不再开发 |

**状态流转**：
```
designing → pending_review → approved/rejected
                          ↓
                        paused
                          ↓
                      cancelled
```

---

### 2.3 开发实现阶段（8状态）

| 状态 | 英文 | 说明 | 触发条件 |
|------|------|------|---------|
| 开发中 | developing | 开始开发 | 设计approved |
| 待测试 | pending_testing | 开发完成 | Dev完成编码 |
| 测试中 | testing | 开始测试 | QA开始测试 |
| 待审核 | pending_review | 测试完成 | QA完成测试 |
| 已批准 | approved | 审核通过 | PM批准 |
| 已拒绝 | rejected | 审核拒绝 | PM拒绝 |
| 已暂停 | paused | 暂停开发 | 技术问题 |
| 已取消 | cancelled | 取消开发 | 不再开发 |

**状态流转**：
```
developing → pending_testing → testing → pending_review → approved/rejected
                                                            ↓
                                                          paused
                                                            ↓
                                                        cancelled
```

---

### 2.4 发布交付阶段（6状态）

| 状态 | 英文 | 说明 | 触发条件 |
|------|------|------|---------|
| 发布中 | publishing | 开始发布 | 开发approved |
| 已发布 | published | 发布成功 | 发布完成 |
| 已暂停 | paused | 暂停发布 | 发布问题 |
| 已取消 | cancelled | 取消发布 | 不再发布 |
| 已删除 | deleted | 删除发布 | 下架产品 |
| 归档 | archived | 归档保存 | 历史版本 |

**状态流转**：
```
publishing → published → archived
          ↓
        paused
          ↓
      cancelled
          ↓
       deleted
```
