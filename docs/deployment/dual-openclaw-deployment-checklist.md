## 📋 部署前确认清单

### 环境要求（两个环境都需要）

#### 硬件要求
- [ ] CPU：≥2 核
- [ ] 内存：≥4GB
- [ ] 存储：≥10GB 可用空间
- [ ] 网络：能访问互联网

#### 软件要求
- [ ] Node.js：v18+（检查：`node -v`）
- [ ] npm：v8+（检查：`npm -v`）
- [ ] Git：已安装（检查：`git --version`）

#### 网络要求
- [ ] 能访问 `https://memos.openmem.net`
- [ ] 防火墙允许 HTTPS 出站（443 端口）
- [ ] 无需公网 IP（MemOS 是云端服务）

## 🔑 关键配置

### 环境变量（.env 文件）
```bash
# MemOS API Key（两个环境必须相同）
MEMOS_API_KEY=mpg-your_key_here

# 共享 user_id（两个环境必须相同）
MEMOS_USER_ID=openclaw-dual-deployment
```

### 工作目录
```bash
# 家里 NAS
~/.openclaw-home/

# 公司服务器
~/.openclaw-work/
```

### 端口配置
```bash
# 家里 NAS：3000
# 公司服务器：3000（或自定义）
```

## ⚠️ 风险与应对

### 风险 1：环境不符合要求
- **应对**：提前确认，必要时升级硬件或更换环境

### 风险 2：网络连通性问题
- **应对**：检查防火墙，确认 DNS 配置

### 风险 3：MemOS 服务不可用
- **应对**：联系 MemOS 支持，或使用备用方案

### 风险 4：部署时间超出预期
- **应对**：分阶段部署，先完成一个环境验证
