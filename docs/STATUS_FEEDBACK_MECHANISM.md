## 📋 反馈触发条件

### 必须反馈的状态变化

1. ✅ **开发完成** → 反馈成果和Review请求
2. ✅ **Review完成** → 反馈批准/拒绝结果
3. ✅ **发布完成** → 反馈ClawHub ID和链接
4. ✅ **开发停止** → 反馈停止原因和进度
5. ✅ **遇到阻塞** → 反馈问题和求助
6. ✅ **每日回顾** → 反馈今日成果和学习
