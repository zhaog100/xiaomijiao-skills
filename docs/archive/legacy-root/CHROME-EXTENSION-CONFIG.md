## 📦 扩展位置

**扩展目录：** `~/.openclaw/browser/chrome-extension`

---

## 🔧 配置步骤

### 1. 打开Chrome扩展管理
```
chrome://extensions
```

### 2. 启用开发者模式
- 右上角开关：开发者模式 → 启用

### 3. 加载扩展
- 点击"加载已解压的扩展程序"
- 选择目录：`~/.openclaw/browser/chrome-extension`
- 确认加载

### 4. 验证状态
- 扩展列表中应显示"OpenClaw Browser Relay"
- 状态：已启用

---

## 💡 使用方式

### 基本操作
1. 在浏览器中打开要控制的标签页
2. 点击扩展图标（OpenClaw标志）
3. Badge变绿 = 已连接
4. 在QQ中对话，AI可以操作该标签页

### 支持的操作
- 自动浏览网页
- 点击元素
- 提取数据
- 填写表单
- 截图
- 文件下载
