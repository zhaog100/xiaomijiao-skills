**根本原因**:
1. **VMware虚拟机限制**
   - CPU: Intel Core Ultra 7 155H（2核虚拟）
   - 虚拟GPU: VMware SVGA 3D
   - **cpuMathCores=0**（检测失败）
   - 无GPU加速

2. **模型加载超时**
   - 模型大小: 328MB
   - 加载时间: 10-15分钟（CPU模式）
   - 超时时间: 60秒（默认）
   - 实际需要: 5-10分钟

3. **线程分配问题**
   - 修改: threadsPerContext() 强制返回4
   - 效果: 仍超时

**修复尝试记录**:

**实践方法**:
