### 1. MCP 集成配置 ⭐⭐⭐⭐⭐

**文件**：`/home/zhaog/.openclaw/config/mcporter.json`

**配置内容**：
```json
{
  "mcpServers": {
    "qmd": {
      "command": "/home/zhaog/.bun/bin/bun",
      "args": [
        "/home/zhaog/.bun/install/global/node_modules/@tobilu/qmd/src/qmd.ts",
        "mcp"
      ],
      "env": {
        "QMD_COLLECTIONS_PATH": "/home/zhaog/.openclaw/workspace"
      }
    }
  }
}
```

**功能**：
- ✅ qmd 以 MCP 服务器形式运行
- ✅ Agent 可自主调用 qmd 工具
- ✅ 支持 query/vsearch/search/get/multi_get/status

**使用方式**：
```python
# Agent 自主调用示例
def answer_user_question(question):
    # 1. 先调用 qmd 检索相关记忆
    relevant_memories = mcp_call('qmd', 'query', {
        'collection': 'daily-logs',
        'query': question,
        'limit': 5
    })
    
    # 2. 拼接精简上下文
    context = "\n".join([m['content'] for m in relevant_memories])
    
    # 3. 调用大模型（只消费精简上下文）
    answer = llm_call(question, context)
    
    return answer
```

**预期效果**：
- Token 节省：90%（2000 Token → 200 Token）
- 准确率提升：72% → 93%
- 响应速度：提升 50%
