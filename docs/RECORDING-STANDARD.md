**维护**: 小米辣 (PM + Dev) 🌶️

---

## 🎯 核心原则

1. **统一格式** - 所有记录用同一套模板
2. **时间戳** - 每条记录都有精确时间
3. **可追溯** - 每条记录可追溯到具体任务/PR
4. **简洁美观** - 使用 emoji + 表格 + 列表
5. **版权声明** - 所有文件包含 MIT License

---

## 🎨 格式规范

### Emoji 使用

| Emoji | 含义 | 使用场景 |
|-------|------|----------|
| ✅ | 完成 | 已完成任务 |
| ⏳ | 等待 | 审核中/进行中 |
| 💰 | 金钱 | 收益/付款 |
| 📊 | 统计 | 数据/报表 |
| 🧠 | 记忆 | 学习/知识 |
| ⚠️ | 警告 | 踩坑/注意 |
| 🌶️ | 小米辣 | 签名/身份 |

### 表格格式

- 表头用 `|` 分隔
- 对齐线用 `---`
- 内容左对齐（默认）

### 列表格式

- 有序列表用 `1. 2. 3.`
- 无序列表用 `-`
- 任务列表用 `- [ ]`

### 代码块

- 行内代码用 `` `code` ``
- 多行代码用 ```language

### 引用

- 用 `>` 表示引用
- 重要引用加粗 `> **重点**`
