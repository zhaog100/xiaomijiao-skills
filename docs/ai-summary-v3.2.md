## 🎯 核心功能（v3.2.0）

### ⭐ AI 摘要系统 ⭐⭐⭐⭐⭐

**功能**：
1. 从对话中提取关键信息
2. 自动生成摘要
3. 智能评估重要性

**生成文件**：
- `.summary.json`：结构化摘要数据
- `SUMMARY.md`：可读性摘要文档

## 🚀 使用方式

### 方式1：独立使用

```bash
# main 代理
bash ai-summarizer.sh

# research 代理
AGENT_NAME=research bash ai-summarizer.sh
```

### 方式2：集成使用

```bash
# session-memory-enhanced v3.2.0 会自动调用 AI 摘要
bash session-memory-enhanced-v3.2.sh
```

## 💡 技术亮点

### 1. 自动关键词提取
- 基于频率统计
- 提取中文词汇（2-10字）
- 自动去重排序

### 2. 重要性评估
- 关键词 >= 8：high
- 关键词 >= 5：medium
- 关键词 < 5：low

### 3. 多代理支持
- 每个代理独立摘要
- 互不干扰
