## 1. 需求概述

### 1.1 背景
双米粒协作过程中会产生涌现词汇（共享语言），需要系统化分析和管理这些词汇，提升协作效率。

### 1.2 目标
**核心目标**：分析米粒儿和小米辣对话中的涌现词汇，生成词汇报告并集成到协作流程。

**具体目标**：
1. ✅ 提取共享词汇
2. ✅ 生成涌现词汇报告
3. ✅ 每周自动回顾
4. ✅ 更新 MEMORY.md 涌现词汇表

### 1.3 范围
**包含**：
- 对话内容分析
- 共享词汇提取
- 涌现词汇报告生成
- 每周自动回顾
- MEMORY.md 更新

**不包含**：
- 实时词汇分析
- 多语言支持
- 词汇翻译

## 2. 功能需求

### 2.1 核心功能（P0）

#### 功能 1：对话内容分析 ⭐⭐⭐⭐⭐
- **描述**：分析米粒儿和小米辣的对话内容
- **输入**：对话记录（Issue 评论/协作文件）
- **输出**：结构化对话数据
- **验收标准**：
  - [ ] 支持 Issue 评论分析
  - [ ] 支持协作文件分析
  - [ ] 对话分类准确

#### 功能 2：共享词汇提取 ⭐⭐⭐⭐⭐
- **描述**：提取双方都使用的词汇
- **输入**：结构化对话数据
- **输出**：共享词汇列表（含出现次数）
- **筛选规则**：
  - 出现次数≥2 次
  - 排除常用词（the/is/are 等）
  - 词长≥3 个字符
- **验收标准**：
  - [ ] 提取准确率 > 90%
  - [ ] 常用词过滤完整
  - [ ] 支持自定义排除词

#### 功能 3：涌现词汇报告生成 ⭐⭐⭐⭐⭐
- **描述**：生成涌现词汇报告
- **输入**：共享词汇列表
- **输出**：Markdown 格式报告
- **报告内容**：
  - 涌现词汇列表（按频率排序）
  - 词汇分类（协作/技术/工具）
  - 使用趋势分析
  - 新增词汇标记
- **验收标准**：
  - [ ] 报告结构清晰
  - [ ] 词汇分类准确
  - [ ] 支持导出 Markdown

#### 功能 4：每周自动回顾 ⭐⭐⭐⭐
- **描述**：每周自动生成词汇回顾
- **输入**：本周对话记录
- **输出**：周度词汇报告
- **验收标准**：
  - [ ] 每周自动生成
  - [ ] 对比上周变化
  - [ ] 标记新增词汇

### 2.2 辅助功能（P1）

#### 功能 5：MEMORY.md 更新
- **描述**：自动更新 MEMORY.md 涌现词汇表
- **验收标准**：
  - [ ] 自动追加新词汇
  - [ ] 保留历史词汇
  - [ ] 支持版本管理
