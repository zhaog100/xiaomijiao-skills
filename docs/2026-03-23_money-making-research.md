## 1. Bug Bounty（安全赏金）

### 平台对比

| 平台 | 注册 | 项目数 | 赏金范围 | 特点 |
|------|------|--------|---------|------|
| **HackerOne** | hackerone.com | 3000+ | $100-$100K+ | 最大平台，企业多 |
| **Intigriti** | intigriti.com | 500+ | €50-$50K+ | 欧洲为主，社区友好 |
| **Bugcrowd** | bugcrowd.com | 200+ | $50-$20K+ | 美国，Crowdmatch AI匹配 |
| **YesWeHack** | yeswehack.com | 300+ | €50-$10K+ | 法国，欧洲政府项目 |

### 入门路径

1. **注册** — 用 GitHub (zhaog100) 注册 HackerOne + Intigriti
2. **学习资源**:
   - PortSwigger Web Security Academy（免费，系统化）: https://portswigger.net/web-security
   - PayloadsAllTheThings（漏洞payload大全）: https://github.com/swisskyrepo/PayloadsAllTheThings
   - Hacker101（HackerOne官方免费课）: https://www.hackerone.com/hacker101
3. **工具安装**:
   - Burp Suite Community（免费版）
   - Nuclei（ProjectDiscovery，自动化漏洞扫描）
   - Subfinder + httpx（子域名发现）
4. **从VDP开始** — 免费的漏洞披露项目练手，无赏金但有声望
5. **进阶** — 转向付费 bounty 项目

### 预估时间线

| 阶段 | 时间 | 目标 |
|------|------|------|
| 注册+学习 | 1-2周 | 完成PortSwigger基础课程 |
| VDP练手 | 2-4周 | 提交前5个有效漏洞 |
| 付费bounty | 1-2月 | 第一个赏金 |

### 预估收入

- 新手期（1-3月）：$0-$500/月
- 中级（3-6月）：$500-$2000/月
- 高级（6月+）：$2000-$10000+/月

## 2. AI Agent 自动化接单

### 可提供服务

| 服务 | 描述 | 建议定价 |
|------|------|---------|
| GitHub 自动化 | CI/CD、PR Bot、Issue管理 | $100-500/单 |
| Web 爬虫/数据采集 | 定制爬虫、数据清洗 | $50-300/单 |
| AI Agent 开发 | 定制 AI 工作流、MCP 集成 | $200-1000/单 |
| 自动化脚本 | 定时任务、监控报警 | $50-200/单 |
| API 集成 | 第三方 API 对接、数据同步 | $100-500/单 |

### 接单平台

| 平台 | 注册 | 佣金 | 特点 |
|------|------|------|------|
| **Upwork** | upwork.com | 10-20% | 全球最大，项目多 |
| **Fiverr** | fiverr.com | 20% | 产品化服务，适合卖标准化Gig |
| **Freelancer** | freelancer.com | 10% | 竞标制 |
| **国内** | 猪八戒/程序员客栈 | 5-10% | 中文市场，价格低 |

### 执行计划

1. **注册账号** — Upwork + Fiverr（需要VPN/海外身份）
2. **作品集** — 整理 GitHub PR、开源项目、ClawHub 技能
3. **第一批Gig**:
   - Fiverr: "I will build a GitHub automation bot" ($50)
   - Fiverr: "I will create a custom web scraper" ($75)
   - Upwork: 搜索 "automation" "python" "github" 类项目竞标
4. **用 OpenClaw 作为生产力工具** — 自动接单、自动交付

### 难点

- Upwork/Fiverr 可能需要海外支付方式
- 需要积累评价和声望
- 竞争激烈，需要差异化
