## 🤖 自动化方案

### 方案1：完全手动（当前）

**优点**：
- ✅ 灵活控制
- ✅ 内容完整

**缺点**：
- ⚠️ 手动操作
- ⚠️ 可能遗漏

**适用场景**：
- 重要反馈（发布完成）
- 复杂情况（遇到阻塞）

---

### 方案2：半自动（推荐）

**优点**：
- ✅ 部分自动化
- ✅ 灵活调整

**实现**：
```bash
# 自动生成反馈模板
bash scripts/generate_feedback.sh [技能名称] [状态]

# 手动补充内容后发布
gh issue comment [number] --file /tmp/feedback.md
```

**脚本示例**：
```bash
#!/bin/bash
# scripts/generate_feedback.sh

SKILL_NAME=$1
STATUS=$2
ISSUE_NUMBER=$3

case $STATUS in
  "completed")
    TEMPLATE="docs/feedback_templates/completed.md"
    ;;
  "reviewed")
    TEMPLATE="docs/feedback_templates/reviewed.md"
    ;;
  "published")
    TEMPLATE="docs/feedback_templates/published.md"
    ;;
  *)
    echo "Unknown status: $STATUS"
    exit 1
    ;;
esac

# 生成反馈内容
cat $TEMPLATE | \
  sed "s/\[技能名称\]/$SKILL_NAME/g" | \
  sed "s/\[number\]/$ISSUE_NUMBER/g" \
  > /tmp/feedback.md

echo "反馈已生成: /tmp/feedback.md"
echo "请手动补充内容后执行: gh issue comment $ISSUE_NUMBER --file /tmp/feedback.md"
```
