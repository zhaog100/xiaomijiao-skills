# Creator Fund Revenue Sharing Pilot

创作者基金收益分成试点计划

## 任务说明

参与 BoTTube 创作者基金，通过高质量内容获得月度 RTC 奖励。

## 奖池分配

- **Month 1**: 150 RTC（按观看次数）
- **Month 2**: 150 RTC（按观看次数）
- **Month 3**: 200 RTC（观看 + 互动）

## 资格要求

- 10+ 订阅者
- 10+ 视频
- 30 天内活跃
- 无垃圾内容
