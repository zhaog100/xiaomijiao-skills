## 双语音识别系统架构

- 模型：Medium（769MB）
- 准确率：95%（优化后）
- 速度：15秒（加载+识别）
- 离线：完全支持
- 隐私：本地处理

**支持方言**
1. 普通话（95%准确率）
2. 粤语（90%准确率）
3. 吴语（85%准确率）
4. 客家话（80%准确率）
5. 闽南话（80%准确率）
6. 四川话（80%准确率，优化后）

**优化参数**
```python
temperature=0.1  # 降低随机性

best_of=5        # 候选数量
language="zh"    # 中文
initial_prompt="这是关于AI助手米粒儿的对话。常用词汇：技能、能力、功能、帮助、服务。"
```

**使用场景**
- ✅ 需要离线识别
- ✅ 涉及隐私内容
- ✅ 需要方言支持
- ✅ 网络不稳定
- ✅ 不限时长

### 2. 硅基流动云端识别（SenseVoice）

**基本信息**
- API：https://api.siliconflow.cn/v1/audio/transcriptions
- 模型：FunAudioLLM/SenseVoiceSmall
- 准确率：100%
- 速度：6.9秒（网络+API）
- 离线：不支持
- 隐私：云端处理

**API配置**
```python
import requests

api_key = "sk-xxx"
with open("audio.mp3", "rb") as f:
    audio_data = f.read()

response = requests.post(
    "https://api.siliconflow.cn/v1/audio/transcriptions",
    headers={"Authorization": f"Bearer {api_key}"},
    files={"file": ("audio.mp3", audio_data, "audio/mpeg")},
    data={"model": "FunAudioLLM/SenseVoiceSmall"},
    timeout=60
)

print(response.json().get("text", ""))
```

**使用场景**
- ✅ 需要高准确率（100%）
- ✅ 简单对话场景
- ✅ 不涉及隐私
- ✅ 网络稳定
- ✅ 快速响应

## 实战测试结果

**测试用例**："你是谁？你是哪个？你在做啥子？你有啥子技能？"

| 系统 | 输出 | 准确率 | 速度 |

- Whisper原始：准确率90%，多个错误
