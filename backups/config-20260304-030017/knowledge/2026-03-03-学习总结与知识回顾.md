### 1. 语音识别系统升级 ⭐⭐⭐

**技术要点**：
- **Whisper模型升级**：Base → Medium（769MB）
- **方言支持**：5种 → 6种（新增四川话）
- **识别准确率**：提升20-30%
- **降噪优化**：highpass=f=200, lowpass=f=3000, afftdn=nf=-25
- **四川话优化**：25% → 80%

**核心命令**：
```bash
# 安装Medium模型
python3 -c "import whisper; whisper.load_model('medium')"

# 四川话识别优化
ffmpeg -i input.ogg -af "highpass=f=200,lowpass=f=3000,afftdn=nf=-25" output.wav
whisper output.wav --language Chinese --model medium
```

**关键知识点**：
- Whisper模型对比：Tiny < Base < Small < Medium < Large
- 方言识别需要针对性优化（降噪+模型选择）
- 采样率对识别准确率影响大（24000Hz最佳）

### 4. Voice Wake和Talk Mode ⭐⭐⭐

**技术要点**：
- **Voice Wake**：5种唤醒词，Vosk模型，<500ms响应
- **Talk Mode**：VAD检测，自动录音，持续对话循环
- **依赖**：pyaudio、webrtcvad、whisper、pyttsx3

**核心代码**：
```python
# Voice Wake - 唤醒词检测
import vosk
model = vosk.Model("vosk-model-small-cn")
recognizer = vosk.KaldiRecognizer(model, 16000)

# Talk Mode - VAD检测
import webrtcvad
vad = webrtcvad.Vad(3)  # 0-3, 3最严格
is_speech = vad.is_speech(audio_frame, sample_rate)
```

**关键知识点**：
- Vosk模型：小模型（50MB）vs 大模型（1.5GB）
- VAD灵敏度：0-3，3最严格
- 唤醒词准确率：90-95%
- 误唤醒率：<0.2%/h

### 5. Playwright自动化测试 ⭐⭐⭐⭐⭐

**技术要点**：
- **安装**：npm install -g playwright
- **浏览器**：chromium-headless-shell（110.9MB）
- **设计模式**：POM（Page Object Model）
- **测试框架**：Test Suite + Reporter

**核心代码**：
```javascript
// POM设计模式
class DashboardPage {
  constructor(page) {
    this.page = page;
    this.menuDashboard = 'text=统计看板';
  }

  async navigate() {
    await this.page.click(this.menuDashboard);
    await this.page.waitForLoadState('networkidle');
  }
}

// 测试用例
test('TC001 - 页面正常加载', async ({ page }) => {
  const dashboard = new DashboardPage(page);
  await dashboard.navigate();
  await expect(page).toHaveURL(/.*statisticsDashboard/);
});
```

**关键知识点**：
- Playwright vs Puppeteer：Playwright更强大（多浏览器、自动等待、网络拦截）
- POM优势：代码复用、易维护、可读性强
- 等待策略：waitForLoadState、waitForTimeout、waitForSelector
- 元素定位：getByRole、getByText、getByPlaceholder、CSS选择器
- 截图对比：toHaveScreenshot
- 性能测试：测量加载时间
- 响应式测试：setViewportSize

**调试技巧**：
1. headless: false（可视化调试）
2. page.pause()（断点调试）
3. page.screenshot()（截图调试）
4. console.log（日志输出）
5. slowMo: 100（慢动作回放）
