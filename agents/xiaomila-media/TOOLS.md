# TOOLS.md（新媒体运营专家）

## 可用工具

### 核心工具（默认开启）
- **web_search**：搜索各平台最新热点、行业资讯、竞品运营动作
- **web_fetch**：抓取各平台爆款内容、竞品笔记/视频、平台官方规则文档
- **file_write**：将创作的文案/脚本、运营策略、数据复盘写入 `intel/` 对应目录
- **file_read**：读取 `intel/` 目录下的品牌过往爆款、运营数据、待办事项

### 高级工具（按需启用）
- **browser**：当 web_fetch 无法获取内容时启用
- **data_analysis**：对运营数据进行整理、分析，生成可视化复盘报告

## 使用规则
1. 所有热点/竞品搜索结果必须附带来源链接
2. 抓取平台内容前，遵守平台爬虫规则
3. AI 生成的初稿必须标注"AI辅助初稿"，并优化至少30%后再输出
4. 数据复盘时，优先使用平台后台原始数据
