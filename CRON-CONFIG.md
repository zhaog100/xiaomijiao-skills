## 📋 任务内容

### 中午任务（12:00）
每次任务执行以下步骤：

1. **更新索引**
   ```bash
   qmd update
   ```
   - 更新所有collections的索引
   - 处理新增、修改、删除的文件

2. **检查记忆文件**
   - 确认今日记忆文件已创建
   - 路径: memory/YYYY-MM-DD.md

3. **更新向量**（可选）
   ```bash
   qmd embed -f
   ```
   - 仅在向量搜索已启用时执行
   - 当前状态: 待启用（需要编译llama.cpp）

4. **记录结果**
   - 将更新结果写入memory/YYYY-MM-DD.md
   - 记录新增文件、更新数量等

### 晚上任务（23:50）
执行完整的每日回顾与查漏补缺流程：

#### 步骤1: 更新QMD索引
1. 运行 `qmd update` 更新索引
2. 检查memory文件夹，确认今日记忆文件已创建
3. 如有新文件，运行 `qmd embed -f` 更新向量（如向量搜索已启用）
4. 记录更新结果到memory/YYYY-MM-DD.md

#### 步骤2: 每日回顾
1. 阅读今日记忆文件（memory/YYYY-MM-DD.md）
2. 阅读今日知识总结（memory/YYYY-MM-DD-knowledge-summary.md，如存在）
3. 总结核心学习内容和成就

#### 步骤3: 查漏补缺
1. 检查知识库完整性
2. 发现遗漏和问题
3. 补充缺失内容
4. 优化文档结构

#### 步骤4: 完善知识库
1. 更新MEMORY.md（提炼精华，移除过时信息）
2. 创建或更新相关文档
3. 生成今日查漏补缺报告（memory/YYYY-MM-DD-gaps-report.md）

#### 步骤5: 生成报告
1. 记录今日成就
2. 总结经验教训
3. 提出改进建议
4. 更新系统状态
