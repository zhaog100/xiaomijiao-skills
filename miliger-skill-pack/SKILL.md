## 📦 套装内容

本套装包含 7 个精心打造的效率工具技能：

### 1️⃣ Context Manager v2.2.2
**上下文管理器** - 无感自动切换，85% 阈值触发
- 实时监控 Token 使用
- 自动创建新会话并迁移记忆
- 零用户干预

### 2️⃣ Smart Memory Sync v1.0.0
**智能记忆同步** - 主动分析 + 三库联动 + 预防性保护
- 实时分析聊天内容
- MEMORY + QMD + Git 三库同步
- 主动切换会话，预防性保护

### 3️⃣ Smart Model Switch v1.3.0
**智能模型切换** - 根据消息复杂度和文件类型自动选择最优模型
- 文件类型检测
- 自动切换 Flash/Main/Coding/Vision/Complex
- 提升响应质量和效率

### 4️⃣ Quote Reader v1.1.0
**引用前文读取** - 自动检索并理解引用内容
- 支持飞书/QQ/企业微信交互式卡片
- 自动检索引用内容
- 理解上下文关联

### 5️⃣ Image Content Extractor v2.0
**图片内容提取** - 智能识别终端/文档/通用模式
- 自动识别图片类型
- 提取内容生成 Markdown
- 支持多种场景

### 6️⃣ Memory Sync Protocol v1.0.0
**记忆优化协议** - 自动精简 MEMORY.md，集成 QMD 检索
- Token 节省 92.5%
- 准确率提升 29%
- 监控 Token 使用

### 7️⃣ GitHub Bounty Hunter v1.0.0
**GitHub 赏金猎人** - 自动监控 bounty 项目
- 自动接任务
- 开发并提交 PR
- 让 OpenClaw 自己赚钱
