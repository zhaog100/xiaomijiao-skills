## 💡 使用建议

### AI集成方式

**推荐：在SKILL.md中添加AI行为规则**

```markdown
## 🤖 AI行为规则（强制）

**每次回复前必须执行：**

```bash
~/.openclaw/workspace/skills/smart-model-switch/scripts/integrate-check.sh
```

**执行时机：**
1. 准备回复用户消息时
2. 生成回复内容之前
3. 静默执行，无问题不输出

**注意事项：**
- ✅ 无输出 = 检查通过
- ⚠️ 有输出 = 需要提醒
- 🔇 冷却期 = 跳过检测
```

### 测试方法

```bash
# 模拟高上下文测试
TEST_CONTEXT_USAGE=87 scripts/integrate-check.sh

# 查看日志
tail -f ~/.openclaw/logs/ai-proactive-check.log

# 查看状态
cat data/context-state.json | jq .
