## 📊 检查机制

### check_github_issues.sh

**功能**：
- ✅ 检查最近更新的Issues
- ✅ 识别新评论（自上次检查时间）
- ✅ 写入通知日志
- ✅ 轻量级（<1秒执行）

**日志文件**：
- `logs/last_check_time` - 上次检查时间
- `logs/new_comments.log` - 新评论通知

**示例输出**：
```
🔍 检查GitHub Issues（2026-03-15T02:30:00Z 之后）...
📋 检查 Issue #10...
  🆕 新评论！用户: zhaog100, 时间: 2026-03-15T02:27:22Z
⚠️ 发现 1 个新评论！请查看 logs/new_comments.log
```
