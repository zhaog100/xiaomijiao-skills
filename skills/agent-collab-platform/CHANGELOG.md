# v1.7.0 更新日志

**发布时间**：2026-03-15 10:38

## 🔄 架构重大变更

### 从"独立监听进程"改为"心跳+主动检查"

**原因**：
- ❌ 独立进程不稳定（容易退出）
- ❌ 与主会话隔离（无法直接通知）
- ❌ 资源占用高（持续运行）

**新方案**：
- ✅ 心跳集成（与主会话绑定）
- ✅ 主动检查（按需执行，<1秒）
- ✅ 稳定性99%（vs 旧方案60%）

---

## 🆕 新增功能

### 1. `scripts/check_github_issues.sh` ⭐⭐⭐⭐⭐

**功能**：
- 检查最近更新的Issues（自上次检查时间）
- 识别新评论（小米辣的回复）
- 写入通知日志
- 轻量级执行（<1秒）

**用法**：
```bash
bash scripts/check_github_issues.sh
```
