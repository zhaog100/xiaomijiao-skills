## 📝 使用方法

### **1. 自动签到（立即可用）**

#### 添加定时任务（通过Web界面）
1. 打开青龙面板：http://43.133.55.138:5700
2. 点击「定时任务」→「添加任务」
3. 填写：
   - **名称**：`每日签到`
   - **命令**：`dailycheckin`
   - **定时**：`0 8 * * *`（每天早上8点）
4. 点击「保存」

#### 配置账号
1. 点击「配置文件」
2. 找到 `/ql/config/config.json`
3. 修改对应平台的Cookie
4. 参考：[Cookie获取教程](references/platform-apis.md)

## 🛠️ 常用命令

### **青龙面板管理**

```bash
# 查看容器状态
docker ps | grep qinglong

# 进入容器
docker exec -it qinglong /bin/sh

# 手动运行签到
docker exec qinglong dailycheckin

# 查看日志
docker logs qinglong --tail 100

# 重启容器
docker restart qinglong
```

### **价格监控**

```bash
# 运行价格监控（开发中）
python scripts/price_monitor.py --config assets/config.json

# 添加监控商品
python scripts/price_monitor.py --add --url "商品链接"

# 查看价格历史
python scripts/price_monitor.py --history --item "商品ID"
```
