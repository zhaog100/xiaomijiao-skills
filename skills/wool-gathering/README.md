## 🛠️ 高级使用

### **1. 配置定时任务**

```bash
# 进入青龙容器
docker exec -it qinglong /bin/sh

# 添加定时任务（每小时检查价格）
echo "0 * * * * python3 /root/.openclaw/workspace/skills/wool-gathering/scripts/spider_unified.py" >> /ql/data/config/crontab.list
```

### **2. 配置推送通知**

编辑 `assets/config_template.json`：

```json
{
  "push_config": {
    "serverchan_enabled": true,
    "serverchan_key": "SCTxxxxxxxx"
  }
}
```

### **3. 使用真实爬虫**

**京东爬虫**（推荐）：
```python
from jd_price_api_simple import JDPriceSpiderSimple

spider = JDPriceSpiderSimple()
result = spider.get_price('100012043978')
```

**淘宝爬虫**：
```python
from taobao_price_simple import TaobaoPriceSpiderSimple

spider = TaobaoPriceSpiderSimple()
result = spider.get_price('560852934047')
```

## ⚠️ 注意事项

### **合规使用**
- ✅ 自动签到（模拟用户行为）
- ✅ 价格监控（公开信息）
- ✅ 优惠券推送（联盟API）
- ❌ 恶意抢券、爬虫滥用、刷单刷评

### **安全建议**
1. 使用小号测试
2. Cookie定期更新（1-3个月）
3. 不要频繁修改配置
4. 定期检查运行日志

### **淘宝爬虫限制**
- ⚠️ 淘宝API需要真实商品ID
- ⚠️ 可能需要登录Cookie
- ⚠️ 建议使用京东爬虫（更稳定）
