# QMD Manager

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> QMD 知识库管理技能 - 本地知识库的语义搜索和管理

## 🎯 简介

QMD Manager 是一个专为项目管理、软件测试和内容创作设计的本地知识库管理工具，提供强大的语义搜索功能。

### 核心功能

- 📚 **项目管理知识搜索** - 文档、最佳实践
- 🧪 **软件测试知识检索** - 方法论、工具使用
- ✍️ **内容创作知识管理** - 公众号、视频号内容
- 🔍 **本地语义搜索** - BM25 + 向量搜索
- 📖 **文档预览和获取** - 查看和加载文档内容

## 🚀 快速开始

### 1. 检查状态

```bash
/qmd status
```

### 2. 搜索知识

```bash
# 关键词搜索
/qmd 项目管理最佳实践

# 语义搜索
/qmd 软件测试自动化

# 混合搜索（推荐）
/qmd 如何写好测试用例
```

### 3. 获取文档

```bash
# 查看完整文档
/qmd get 0
```

## 🔧 可用工具

### 1. qmd_search - 关键词搜索

**快速关键词搜索（BM25）**

```bash
/qmd 项目管理
```

**参数**：
- `query` - 搜索关键词
- `collection` - 集合名（可选）
- `limit` - 结果数量（默认10）

---

### 2. qmd_vsearch - 语义向量搜索

**语义向量搜索（Embedding）**

```bash
/qmd 如何提高团队协作效率
```

**特点**：
- 理解查询意图
- 支持同义词
- 跨语言搜索
