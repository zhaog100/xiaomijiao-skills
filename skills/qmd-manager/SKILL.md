# QMD 管理器技能 - 项目管理和测试知识管理

## 📋 技能描述
这是一个为项目管理、软件测试和内容创作设计的 QMD (Quick Markdown Search) 技能，用于本地知识库管理和语义搜索。

## 🎯 主要功能
- **项目管理知识搜索** - 搜索项目管理文档、最佳实践
- **软件测试知识搜索** - 搜索测试方法论、工具使用
- **内容创作知识管理** - 管理公众号、视频号内容
- **本地语义搜索** - 使用 BM25 和向量搜索
- **文档预览和获取** - 查看和加载文档内容

## 🔧 可用工具
- **qmd_search** - 快速关键词搜索
- **qmd_vsearch** - 语义向量搜索
- **qmd_query** - 混合搜索和重排
- **qmd_get** - 获取文档内容
- **qmd_status** - 查看索引状态

## 📝 使用方法
```bash
# 搜索项目管理知识
/qmd 项目管理最佳实践

# 搜索软件测试方法
/qmd 软件测试自动化

# 查看知识库状态
/qmd status

# 添加知识库集合
/qmd add ./docs

# 生成嵌入（首次使用需要）
/qmd embed
```

## 🎨 特色功能
- **TUI 浏览器** - 交互式知识浏览
- **多集合管理** - 支持多个知识库
- **LLM 重排** - 智能结果排序
- **文档预览** - 快速查看内容

## 📁 知识库结构建议
```
./knowledge/
├── project-management/    # 项目管理文档
├── software-testing/     # 软件测试文档  
└── content-creation/     # 内容创作文档
```

## ⚙️ 安装说明
1. 安装 QMD 二进制文件：`brew install oven-sh/bun/bun && bun i -g https://github.com/tobi/qmd`
2. 添加知识库集合：`/qmd add ./knowledge`
3. 生成嵌入：`/qmd embed`
4. 开始搜索：`/qmd 您的查询`
