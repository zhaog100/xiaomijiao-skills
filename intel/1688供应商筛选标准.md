# 🏭 1688 供应商筛选标准（小批量代发版）

**创建**: 2026-03-25  
**维护**: 小米辣 🌶️  
**适用**: 跨境电商起步期，小批量测款 + 代发

---

## 🎯 核心筛选原则

> **靠谱供应商 = 能代发 × 小MOQ × 质量稳定 × 沟通顺畅**

---

## ✅ 必须满足（硬指标）

| 指标 | 标准 | 原因 |
|------|------|------|
| **支持代发** | 明确标注"一件代发"或"跨境代发" | 起步期不囤货 |
| **MOQ** | ≤50 件（最好支持 10 件起） | 小批量测款 |
| **店铺年限** | ≥3 年 | 淘汰短命店 |
| **回头率** | ≥30% | 说明老客认可 |
| **发货速度** | 48 小时内发货 | 跨境物流本身慢，源头不能再拖 |

---

## ⭐ 优先选择（加分项）

| 指标 | 标准 | 加分原因 |
|------|------|----------|
| **实力商家** | 带"实力商家"标签 | 1688官方认证，品控有保障 |
| **跨境专供** | 标注"跨境专供"/"外贸专区" | 了解出口标准，标签/包装合规 |
| **提供质检报告** | 有 CE/FCC/FDA 等认证 | Amazon 上架部分品类必须有 |
| **支持定制** | Logo/包装/说明书可定制 | 后期品牌化必须 |
| **产业带商家** | 义乌/深圳/中山/台州等 | 源头工厂价格更低 |
| **7天无理由退换** | 明确标注 | 减少试错成本 |

---

## ❌ 避坑红线（碰到就跳过）

| 红线 | 说明 |
|------|------|
| 🚫 **无实物图/全用渲染图** | 大概率货不对板 |
| 🚫 **价格远低于同行** | 偷工减料或钓鱼 |
| 🚫 **评价大量差评** | 尤其看"质量差""发错货""不回消息" |
| 🚫 **不接受退换** | 出问题你全部承担 |
| 🚫 **无法提供认证** | 目的国清关/上架会被卡 |
| 🚫 **旺旺长期不在线** | 后续售后沟通成本极高 |

## 📋 供应商评估打分表

每家供应商用这个表打分，**≥7分** 才列入候选。

| 维度 | 权重 | 1-10 分标准 |
|------|------|-------------|
| 价格竞争力 | 20% | 同品类前30%=8+，中等=5-7 |
| 产品质量 | 25% | 看评价+买样品验证 |
| 代发能力 | 15% | 一件代发=10，50件起=6 |
| 沟通响应 | 15% | 2小时内回复=9+，半天=5 |
| 定制能力 | 10% | 支持全定制=10，仅换标=6 |
| 认证齐全 | 10% | CE+FCC+FDA=10，无认证=1 |
| 售后保障 | 5% | 7天退换=10，不退=1 |

## 🔍 实操搜索技巧

### 搜索关键词组合
```
品类名 + 跨境     例：收纳盒 跨境
品类名 + 外贸     例：宠物碗 外贸
品类名 + 一件代发  例：LED灯 一件代发
品类名 + 亚马逊   例：瑜伽带 亚马逊
```

### 筛选步骤
1. 搜索品类关键词
2. 左侧筛选：经营模式→生产加工
3. 排序：按"综合排序"或"成交额"
4. 勾选"实力商家"
5. 点进店铺看：年限/回头率/评价
6. 旺旺联系，问这几个问题 👇

### 必问清单（直接复制发给供应商）
```
你好，我做跨境电商，想了解下：
1. 支持一件代发吗？发货时效多久？
2. 最低起订量多少？
3. 能提供 CE/FCC 等认证吗？
4. 支持贴牌/定制包装吗？MOQ多少？
5. 有跨境客户案例吗？主要发哪些国家？
6. 质量问题怎么处理？支持退换吗？
7. 能提供产品实拍图和视频吗？
```

## 🚀 建议执行流程

```
第1天：每个品类搜10家 → 初筛留5家
    ↓
第2天：旺旺沟通 → 问必问清单 → 淘汰不合格
    ↓
第3天：每个品类留2-3家 → 下样品单（每家2-3款）
    ↓
第5-7天：收到样品 → 实物对比 → 打分评估
    ↓
第7天：确定每个品类 1 家主力 + 1 家备选
    ↓
第8天起：首批小量测款上架
```

**预算估算：** 3个品类 × 3家 × 3款样品 ≈ 27 个样品，费用约 ¥500-1,000
