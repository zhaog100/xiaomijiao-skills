## 💡 核心逻辑

```
1688 供货（零库存代发）
    ↓

```

**优势**: 零开店费 + 零库存 + 零佣金 + 免费流量

## 第一步：1688 选品建库（Day 1）

### 操作
1. 按之前的筛选标准，每个品类选 3-5 家供应商
2. 建立**供应商清单表**：

| 产品 | 供应商 | 采购价 | 发货时效 | 备注 |
|------|--------|--------|----------|------|
| 猫咪自嗨球 | XX店 | ¥12 | 48h | CE/FCC认证 |
| ... | ... | ... | ... | ... |

3. 保存供应商的**高清产品图/视频**（小红书素材用）
4. 联系供应商确认：代发时不放名片/广告

### 注意
- ⚠️ 要求供应商发货**不带任何宣传物料**
- ⚠️ 确认支持**写闲鱼买家地址直发**

## 第二步：小红书种草号搭建（Day 1-2）

### 账号定位（选一个）

| 定位方向 | 适合品类 | 内容风格 |
|----------|----------|----------|
| 🐱 养猫日常 | 宠物用品 | "我家猫主子的新玩具" |
| 🏠 家居好物分享 | 收纳/灯具 | "租房改造/好物推荐" |
| 💡 氛围感生活 | LED灯/收纳 | "小成本提升幸福感" |

**推荐先做"养猫日常"**— 宠物内容天然高互动

### 内容模板（每篇 3-5 分钟搞定）

**爆款标题公式：**
```
数字 + 痛点/好奇 + 品类词
例：
- "养猫3年，这个¥19的玩具让我家猫疯了"
- "猫咪自己玩2小时不用管！懒人养猫神器"
- "别再买逗猫棒了！这个球它能自己玩一整天"
```

**内容结构：**
```
封面：猫咪玩球的动图/照片（吸引点击）
正文：
① 痛点引入（猫太无聊拆家/上班没空陪玩）
② 产品展示（功能/外观/猫的反应）
③ 真实体验（优缺点都说，增加可信度）
④ 引导私信（"想要链接的姐妹扣1"）
```

### 发布节奏
- 每天 1-2 篇笔记
- 最佳发布时间：12:00-14:00 / 18:00-20:00 / 21:00-23:00
- 前 7 天不要带任何广告，先养号

### 引流话术（评论区/私信）
```
❌ 禁止说："闲鱼搜XXX" （会被封）
✅ 正确说法：
- "私信我发你链接～"
- "戳我主页有联系方式"
- "小窗我告诉你在哪买"
→ 然后私信发闲鱼链接
```

## 第三步：闲鱼上架成交（Day 2）

### 上架技巧

**标题公式：**
```
品类词 + 卖点 + 场景词
例："智能猫咪玩具球 自动逗猫 上班族解放双手 电动自嗨球"
```

**定价公式：**
```
售价 = 1688采购价 × 2~3 倍
例：采购¥12 → 闲鱼卖¥25-35
利润：¥13-23/单
```

**详情页要点：**
- 用小红书同款图片（保持风格一致）
- 写清楚：功能、尺寸、颜色可选、发货时效
- 标注"全新未拆封""厂家直发"
- 设置"我想要"自动回复

### 闲鱼 SEO
- 每天擦亮（刷新）1 次
- 多上不同款式/组合（10-20 个链接）
- 标题覆盖不同关键词

## ⚠️ 避坑指南

1. **小红书绝不直接放链接** — 会被限流/封号
2. **闲鱼不要刷单** — 封号没商量
3. **供应商选包邮的** — 否则运费吃利润
4. **售后提前准备话术** — 退货让买家退到供应商
5. **同一产品多个链接** — 不同标题/图片，覆盖更多搜索词
6. **小红书素材要原创** — 可以参考但不能搬运
