## 💡 核心知识点

### 1. Cookie管理
```
Cookie获取步骤：
1. 登录平台网页版
2. F12打开开发者工具
3. Application > Cookies
4. 找到关键Cookie值
5. 添加到青龙面板环境变量

注意事项：
- Cookie有效期（B站30天，小红书7-15天）
- Cookie安全（不要泄露）
- 定期更新（失效后重新获取）
```

### 2. API调用
```
B站API：
- 用户信息: https://api.bilibili.com/x/web-interface/nav
- 签到: https://api.bilibili.com/x/web-interface/coin/today/add
- 投币: https://api.bilibili.com/x/web-interface/coin/add

小红书API：
- 需要抓包分析
- 可能有反爬机制
- API可能变化

技术要点：
- 使用requests库
- 添加正确的请求头
- 处理JSON响应
- 错误处理
```

### 3. 反爬机制
```
常见反爬：
1. User-Agent检测
   - 添加真实User-Agent

2. Referer检测
   - 添加正确的Referer

3. Cookie验证
   - 保持Cookie有效

4. 频率限制
   - 控制请求频率
   - 不要过于频繁

5. 设备指纹（拼多多）
   - 需要特殊处理
   - 更复杂
```
