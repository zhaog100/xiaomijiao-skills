# memory/ 日志规范 v1.0

## 📁 命名规则
- **每日日志**: `YYYY-MM-DD.md`（唯一，当天所有记录合并到这一个文件）
- **归档**: 当天结束后移到 `archive/YYYY-MM-DD.md`

## 📝 每日日志格式（模板）

```markdown
# YYYY-MM-DD 日志

## 📊 概览
| 指标 | 数据 |
|------|------|
| 时段 | HH:MM - HH:MM |
| PR/任务 | N 个 |
| 学习 | N 篇 |
| 关键词 | 2-3个 |

## ✅ 完成
- **[项目] #N — 标题**（一句话结果）

## 📚 学习
- **标题**: 核心收获一句话

## ⚠️ 教训
- [教训]: 原因 → 方案

## 💡 待办
- [ ] 明天/后续
```

## 📏 规则
- **上限**: 50行，超过就精简（只留关键，不留过程）
- **无过程描述**: 不写"做了什么"，写"结果是什么"
- **无大段代码**: 代码链接或文件路径即可
- **无重复**: MEMORY.md已有的不重复写
- **归档**: `mv memory/YYYY-MM-DD.md memory/archive/YYYY-MM-DD.md`
