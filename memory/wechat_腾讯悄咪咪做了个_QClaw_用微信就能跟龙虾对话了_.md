# 腾讯悄咪咪做了个 QClaw，用微信就能跟龙虾对话了！

**发布时间**: 特工小天

## 文章内容

内容编辑丨特工小天

内容审核丨特工少女

上门安装的天塌了...！

据悉，腾讯最近在研发一款 OpenClaw 一键启动包的产品，QClaw。

下载 QClaw 应用后，即可在本地电脑上一键轻松部署小龙虾。注意是本地，不是云端。如果你本地有装过龙虾，还可以直接一键关联。

模型支持 QClaw 混合路由模型（国内几家热门的拼盘）和自定义模型。

除了一键部署之外，还可以一键直连微信。也就是说，除了飞书、钉钉、QQ 等 IM 之外，现在你可以在你的个人微信上，跟你的龙虾通信了。

关联后将在 QClaw 客服这里沟通。

有非常多刚部署好小龙虾的小伙伴，并不知道该怎么养，怎么使用。值得一提的是，QClaw 预制几个实用的操作，比如远程操控电脑、手机远程办公、社媒自动运营涨粉、GitHub 项目自动开发。

比如让他来发小红书笔记，去热门帖子下面互动引流。

比如让它到观猹上找一些 Claw 类的产品，和发布 Claw 类内容的博主，然后帮我点上关注。

登录后，龙虾成功找到并关注上了，还给了一个总结。

再比如点击 GitHub 项目自动开发之后，他真的本地拉起了浏览器开始登录 GitHub，建立仓库，提交代码。

目前产品还处于内测阶段，预计近段时间内将上线。邀请码可以在观猹平台或观猹社群中蹲，开放后将第一时间通知大家。

*公众号后台回复关键词「龙虾」，加入 🦞龙虾养殖交流群。
