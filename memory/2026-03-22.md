## ⚠️ 踩坑记录

### 1. GitHub clone 超时

**问题**: VPS 到 GitHub 网络不稳定，git clone 反复超时

**解决**: 用 GitHub API 直接创建 blob→tree→commit→ref

```bash
# 获取文件 SHA
gh api repos/OWNER/REPO/contents/PATH --jq '.sha'
# 创建 blob
CONTENT=$(base64 -w 0 file)
BLOB=$(gh api repos/OWNER/REPO/git/blobs -f "content=$CONTENT" -f encoding=base64 --jq '.sha')
# 创建 tree/commit/branch...
```

### 2. JSON Schema oneOf 歧义

**问题**: `oneOf: [{type: array}, {description: "any"}]` 导致 node link 匹配两个分支

**解决**: 改为 `additionalProperties: true`，不限制 inputs 值类型

### 3. github-bounty-hunter.sh 语法错误

**问题**: 行 265 gitcoin) 在 esac 外面

**解决**: 移到 case 语句内部
