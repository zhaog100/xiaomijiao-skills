## 📚 今日学习内容

### 1. 技术学习

#### WebSocket长连接

**核心概念**：
- ✅ WebSocket vs HTTP回调
- ✅ 实时双向通信
- ✅ 无需公网IP
- ✅ 自动重连机制

**实现方式**：
```python
import lark_oapi as lark

# 创建WebSocket客户端
ws_client = lark.ws.Client(
    app_id=APP_ID,
    app_secret=APP_SECRET,
    event_handler=dispatcher,
    log_level=lark.LogLevel.INFO
)

# 启动长连接
ws_client.start()
```

**优势**：
- ✅ 实时性更强（毫秒级延迟）
- ✅ 代码更简洁（lark-oapi原生支持）
- ✅ 无需端口映射（飞书主动连接）

---

#### lark-oapi SDK使用

**核心API**：
- `lark.ws.Client` - WebSocket长连接客户端
- `EventDispatcherHandler` - 事件分发器
- `register_p2_im_message_receive_v1` - 注册消息事件

**代码示例**：
```python
# 创建事件分发器
dispatcher = lark.EventDispatcherHandler.builder(
    "",  # encrypt_key
    "",  # verification_token
).register_p2_im_message_receive_v1(handle_message).build()

# 处理消息
def handle_message(event):
    message = event.event.message
    sender = event.event.sender
    content = message.content
    sender_open_id = sender.sender_id.open_id
```
