**版本**: v1.0.0
**适配设备**: Redmi K40 Gaming (Hyper OS)

## 快速开始

### 1. 下载安装

**Auto.js Pro（免费版）下载地址**：
- 官网：http://www.autojs.org/
- 蓝奏云：https://wwx.lanzoui.com/b01n6h62b
  密码: auto

**安装步骤**：
```
1. 下载Auto.js APK
2. 允许安装未知来源应用
3. 安装并打开
4. 授予无障碍服务权限
```

### 2. 导入脚本

**步骤**：
```
1. 打开Auto.js
2. 点击右下角「+」
3. 选择「导入脚本」
4. 选择脚本文件：
   - douyin_auto_sign.js（抖音极速版）
   - kuaishou_auto_sign.js（快手极速版）
```

### 3. 运行脚本

**抖音极速版**：
```
1. 打开Auto.js
2. 点击「抖音极速版自动签到」
3. 点击「运行」按钮
4. 等待自动操作完成
5. 复制显示的Cookie
6. 发送给米粒管家
```

**快手极速版**：
```
同上操作
```

## 常见问题

### Q1: 提示"无障碍服务未开启"？
```
解决：
设置 → 无障碍 → Auto.js → 开启
```

### Q2: 脚本运行失败？
```
解决：
1. 确认已授予所有权限
2. 确认APP已安装
3. 手动打开一次APP
```

### Q3: 无法点击签到按钮？
```
解决：
1. 手动点击一次签到
2. 重新运行脚本
3. 更新脚本版本
```
