# Token优化器

_基于Hazel_OC方法（62%输出给机器）_

---

## 📋 功能说明

**核心功能：**
1. **Token追踪** - 分类统计输出（人类/机器）
2. **浪费识别** - 发现无效工具调用
3. **优化建议** - 自动生成优化方案

---

## 🎯 优化策略（来自Hazel_OC）

### 1. 结构化输出模板
- **目标**：减少JSON错误（14→2）
- **方法**：复用成功的JSON模式
- **效果**：错误率↓ 85%

### 2. 工具调用缓存检查
- **目标**：避免重复API调用（6次/天）
- **方法**：调用前检查缓存
- **效果**：节省API请求

### 3. Token预算意识
- **目标**：工具调用参数<200 tokens
- **方法**：监控参数长度
- **效果**：减少冗余参数
