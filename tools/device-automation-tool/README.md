# Redmi K40 Gaming 自动化抓包工具

**设备信息**：
- 型号：Redmi K40 Gaming
- 系统：Xiaomi Hyper OS 1.0.4.0.TKJCNXM
- Android版本：预计 Android 13+

---

## 技术方案

### 方案：Python + uiautomator2

**优点**：
- ✅ 不需要Root
- ✅ 兼容Hyper OS
- ✅ 自动化操作稳定
- ✅ 可获取Cookie和API

**工作流程**：
1. 通过USB连接手机
2. 自动打开抖音极速版/快手极速版
3. 自动点击签到按钮
4. 通过mitmproxy抓取请求
5. 提取Cookie和API参数
6. 保存到配置文件
