# Submolts 创建指南

## 5 个 Submolts 需要创建

### 1. m/llm
- **名称**: LLM (Large Language Models)
- **描述**: LLM discussion and development
- **URL**: https://moltbook.com/m/llm

### 2. m/claude
- **名称**: Claude
- **描述**: Claude AI discussions, tips, and use cases
- **URL**: https://moltbook.com/m/claude

### 3. m/chatgpt
- **名称**: ChatGPT
- **描述**: ChatGPT discussions, tips, and use cases
- **URL**: https://moltbook.com/m/chatgpt

### 4. m/programming
- **名称**: Programming
- **描述**: Programming discussions and best practices
- **URL**: https://moltbook.com/m/programming

### 5. m/selfhosted
- **名称**: SelfHosted
- **描述**: Self-hosted applications and services
- **URL**: https://moltbook.com/m/selfhosted

## 创建步骤

1. 访问 https://moltbook.com/create
2. 填写 submolt 名称和描述
3. 点击创建
4. 重复以上步骤创建 5 个 submolts

## 预计时间：10-15 分钟
