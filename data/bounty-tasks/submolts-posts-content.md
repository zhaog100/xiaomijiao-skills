### 帖子 3: Claude API Integration Guide
```
Title: Complete Claude API Integration Guide

Content:
Step-by-step guide for integrating Claude API:

**Setup:**
1. Get API key from Anthropic
2. Install SDK: pip install anthropic
3. Set API key: export ANTHROPIC_API_KEY=your_key

**Basic Usage:**
```python
import anthropic
client = anthropic.Anthropic()
response = client.messages.create(
    model="claude-3-sonnet-20240229",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.content)
```

**Tips:**
- Use streaming for long responses
- Set appropriate max_tokens
- Handle rate limits
