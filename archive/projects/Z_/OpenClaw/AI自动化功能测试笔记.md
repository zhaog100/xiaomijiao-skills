# AI自动化功能测试笔记

## 1. 测试背景
- 项目无专职测试人员
- 利用AI进行代码编写和自动化测试，提高效率

## 2. 工具与方法
- 核心工具：Playwright、MCP、Claude Code
- 利用自然语言指令指导AI进行测试

## 3. 关键流程
- 学习网站功能，罗列模块
- 生成详细测试用例
- 准备测试数据
- 按用例执行测试，截图记录，生成报告

## 4. 实战效果
- 154个测试用例，覆盖11个模块
- 测试时长约3小时，覆盖率97.4%
- 详细测试报告，发现4个问题

## 5. 思维、工具、方法
- 思维方式：迭代探索、问题导向、人机协作、规划优先执行
- 工具：Playwright、MCP、Claude Code、截图引导
- 方法：功能学习、分步规划、循环测试、截图记录
