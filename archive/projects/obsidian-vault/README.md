# Obsidian Vault

这是一个 Obsidian vault，用于存储和组织笔记。

## 特性

- ✅ 纯文本 Markdown 格式
- ✅ 支持双向链接 `[[]]`
- ✅ 支持标签 `#tag`
- ✅ 支持附件和图片

## 使用方式

1. **直接编辑**：使用任何文本编辑器编辑 `.md` 文件
2. **QMD 检索**：使用 `qmd search` 搜索内容
3. **OpenClaw 集成**：通过 OpenClaw 自动管理笔记

## 与 OpenClaw 集成

- Session-Memory Enhanced：自动保存会话记忆
- QMD Manager：自动索引和检索
- Playwright Scraper：从网页导入笔记
