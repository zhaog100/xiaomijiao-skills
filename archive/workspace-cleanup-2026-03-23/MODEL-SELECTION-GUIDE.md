## 🔄 模型路由策略

### 策略1：按任务复杂度

```
简单任务（<100 tokens预期输出）
→ glm-4.7-flash
→ 速度：<1秒

中等任务（100-500 tokens）
→ glm-5（当前）
→ 速度：1-2秒

复杂任务（500+ tokens）
→ qwen3-max
→ 速度：2-5秒
```

### 策略2：按任务类型

```
闲聊/问候
→ glm-4.7-flash
→ 快速响应，节省成本

代码/技术
→ coding-glm-5-free
→ 代码优化，技术理解

文档/写作
→ qwen3.5-plus
→ 中文优化，文档生成

分析/推理
→ glm-5 或 qwen3-max
→ 深度分析，逻辑推理
```

### 策略3：按时间/成本

```
白天（工作时段）
→ glm-5
→ 平衡性能

深夜（闲时）
→ qwen3-max
→ 最佳质量

成本敏感
→ glm-4.7-flash
→ 最快最省
```
