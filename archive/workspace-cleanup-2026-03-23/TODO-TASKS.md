```
# Git 更新策略
1. 每次更新前先检查是否冲突
   git fetch origin
   git diff master origin/master --stat

2. 以合并为主（优先保留双方内容）
   git pull origin master --no-rebase
   
3. 如果有冲突，手动解决后提交
   git add .
   git commit -m "merge: 解决冲突，保留双方内容"

4. 最后推送
   git push origin master
```
