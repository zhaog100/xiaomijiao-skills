## 📋 日常维护建议

### 每周清理
```bash
# 清理临时文件
rm -rf /tmp/*

# 清理npm缓存
npm cache clean --force

# 清理系统日志（保留3天）
journalctl --vacuum-time=3d
```

### 每月检查
```bash
# 检查磁盘使用
df -h /

# 检查大文件
du -sh /root/.cache/* | sort -rh | head -10

# 清理旧日志
find /root/.openclaw/workspace/logs -name "*.log" -mtime +30 -delete
```

### 监控告警
- 磁盘使用率 > 70%：警告
- 磁盘使用率 > 85%：紧急清理
