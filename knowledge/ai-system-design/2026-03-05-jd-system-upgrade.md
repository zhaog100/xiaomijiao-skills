## 技术细节

### Crontab格式规范

**正确格式**：
```bash
log_name=scripts_jd_wrapper task /ql/scripts/jd_wrapper.sh node jd_script.js
```

**关键点**：
- 必须包含log_name前缀
- log_name格式：scripts_jd_wrapper
- 确保脚本路径正确
- 使用node命令执行.js文件

### Docker网络配置

**DNS配置**：
```bash
docker run --dns 8.8.8.8 --dns 114.114.114.114 ...
```
