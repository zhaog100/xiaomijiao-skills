## 📱 小红书种草

### 账号定位

| 定位 | 说明 |
|------|------|
| **垂直领域** | 专注一个类目 |
| **人设打造** | 真实用户分享 |
| **内容价值** | 有用、有趣、有共鸣 |

### 内容创作

| 类型 | 说明 | 频率 |
|------|------|------|
| **产品测评** | 真实使用体验 | 3 篇/周 |
| **使用教程** | 产品使用方法 | 2 篇/周 |
| **对比评测** | 多款产品对比 | 1 篇/周 |
| **生活分享** | 日常使用场景 | 2 篇/周 |

### 爆款标题公式

```
痛点 + 解决方案 + 效果
例：皮肤暗沉怎么办？用这个 7 天白了 2 个度！
```

```
人群 + 场景 + 产品
例：上班族必备！办公室 5 分钟快速补妆神器
```

```
数字 + 效果 + 时间
例：30 天瘦 10 斤！这个减肥方法太有效了
```

### 引流技巧

1. **评论区互动** - 回复用户评论
2. **私信引导** - 引导到闲鱼
3. **主页留联系方式** - 委婉留微信
4. **创建合集** - 整理相关产品

### 账号设置

| 项目 | 说明 |
|------|------|
| **头像** | 真实头像或品牌 Logo |

| **简介** | 说明主营产品和优势 |
| **实名认证** | 必须认证增加信任 |

### Listing 优化

| 要素 | 优化技巧 |
|------|----------|
| **标题** | 包含关键词、品牌、型号 |
| **图片** | 9 张图，实拍 + 细节 + 场景 |
| **描述** | 详细参数、使用感受、售后说明 |
| **价格** | 参考同行，略低一点 |
| **标签** | 添加 5-10 个相关标签 |
