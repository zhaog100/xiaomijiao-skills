# 🚢 出口流程详解

_从接单到收汇的完整出口流程_

**版本**: v1.0  
**更新**: 2026-03-22

---

## 📋 出口流程概览

```
接单 → 备货 → 订舱 → 报关 → 装船 → 制单 → 结汇 → 核销
```

---

## 1️⃣ 接单

### 收到客户订单

**关键文件**:
- [ ] Purchase Order (PO) - 采购订单
- [ ] Proforma Invoice (PI) - 形式发票

**确认事项**:
- [ ] 产品规格、数量、价格
- [ ] 贸易术语 (FOB/CIF 等)
- [ ] 付款方式 (T/T、L/C 等)
- [ ] 交货期
- [ ] 包装要求

### 形式发票模板

```
PROFORMA INVOICE

No.: PI20260322-001
Date: March 22, 2026

Seller: [公司名称]
Buyer: [客户名称]

Product: [产品名称]
Quantity: [数量]
Unit Price: USD [单价] [贸易术语]
Total Amount: USD [总金额]

Payment: [付款方式]
Shipment: [交货期]
Port of Loading: [装运港]
Port of Destination: [目的港]

Bank Info:
  Bank Name: [银行名称]
  Account No: [账号]
  SWIFT Code: [代码]
```

---

## 2️⃣ 备货

### 生产/采购

- [ ] 下达生产订单
- [ ] 确认原材料
- [ ] 跟进生产进度
- [ ] 质量检验

### 包装

- [ ] 内包装（产品包装）
- [ ] 外包装（运输包装）
- [ ] 唛头（Shipping Mark）
- [ ] 装箱单准备

---

## 3️⃣ 订舱

### 联系货代订舱

**提供信息**:
- [ ] 货物信息（品名、重量、体积）
- [ ] 装运港/目的港
- [ ] 预计出货日期
- [ ] 贸易术语

**获取文件**:
- [ ] Booking Confirmation - 订舱确认书
- [ ] Shipping Order (S/O) - 装货单

---

## 4️⃣ 报关

### 准备报关单据

| 单据 | 用途 | 数量 |
|------|------|------|
| 合同 | 贸易证明 | 1 份 |
| 发票 | 货值证明 | 3 份 |
| 装箱单 | 货物明细 | 3 份 |
| 报关委托书 | 委托报关 | 1 份 |
| 报关单 | 海关申报 | 1 份 |

### 报关流程

1. 提交报关单据给报关行
2. 报关行向海关申报
3. 海关查验（如需要）
4. 海关放行
5. 获取放行单

---

## 5️⃣ 装船

### 货物装运

- [ ] 送货到码头/仓库
- [ ] 装集装箱
- [ ] 获取场站收据
- [ ] 换取提单

### 提单类型

| 类型 | 全称 | 说明 |
|------|------|------|
| **B/L** | Bill of Lading | 海运提单 |
| **Sea Waybill** | 海运单 | 不可转让 |
| **Air Waybill** | 空运单 | 空运使用 |

---

## 6️⃣ 制单

### 准备结汇单据

**核心单据**:
- [ ] Commercial Invoice - 商业发票
- [ ] Packing List - 装箱单
- [ ] Bill of Lading - 提单
- [ ] Certificate of Origin - 原产地证
- [ ] Insurance Policy - 保险单（CIF 术语）

**其他可能需要的单据**:
- [ ] 质量检验证书
- [ ] 熏蒸证书
- [ ] 卫生证书

---

## 7️⃣ 结汇

### T/T 电汇

**流程**:
1. 发货后发送提单副本给客户
2. 客户支付尾款
3. 收到款项后寄送正本单据

### L/C 信用证

**流程**:
1. 客户开立信用证
2. 审核信用证条款
3. 按信用证要求备货装运
4. 提交单据给银行
5. 银行审核单据
6. 付款/承兑

---

## 8️⃣ 核销

### 出口收汇核销

- [ ] 登录外汇管理局系统
- [ ] 录入收汇信息
- [ ] 提交核销单据
- [ ] 完成核销

### 出口退税

**所需材料**:
- [ ] 出口货物报关单
- [ ] 出口发票
- [ ] 进货发票
- [ ] 收汇核销单

**退税率**: 根据商品 HS 编码确定（通常 5%-17%）

---

## 📊 时间轴

| 阶段 | 时间 | 负责方 |
|------|------|--------|
| 接单 | Day 1-3 | 业务 |
| 备货 | Day 4-20 | 生产 |
| 订舱 | Day 18-22 | 货代 |
| 报关 | Day 23-24 | 报关行 |
| 装船 | Day 25 | 码头 |
| 制单 | Day 25-26 | 单证 |
| 结汇 | Day 27-30 | 财务 |
| 核销 | Day 31-35 | 财务 |

---

## ⚠️ 注意事项

1. **单证一致**: 所有单据信息必须一致
2. **时间节点**: 严格控制每个环节时间
3. **质量把控**: 出货前必须检验
4. **风险预防**: 购买出口信用保险
