## 📋 起号方案（第 1-2 周）

### 1️⃣ 账号定位

| 维度 | 建议 |
|------|------|
| **赛道选择** | 家居好物 / 数码配件 / 个护美妆（三选一） |

## 📝 爆款笔记公式

### 标题公式（5 种）

| 公式 | 示例 |
|------|------|
| **数字 + 痛点** | "5 个让租房秒变温馨的小物件" |
| **反转 + 惊喜** | "9.9 买的东西，朋友以为花了 200" |
| **身份 + 推荐** | "在台湾生活 10 年，这些好物回购无数次" |
| **场景 + 解决方案** | "出差党必备！行李箱收纳神器" |
| **对比 + 测评** | "花 500 vs 花 50，效果居然一样？" |

### 内容结构

```
封面图（决定点击率 80%）
├── 高质量产品图 / 使用场景图
├── 大字标题（痛点/好奇心）
└── 干净背景，色彩统一

正文（决定互动率）
├── 第 1 段：痛点引入（为什么买？）
├── 第 2 段：产品体验（真实感受）
├── 第 3 段：使用效果（前后对比）
├── 第 4 段：购买建议（适合谁？）
└── 结尾：互动引导（你们觉得呢？）

标签（决定曝光量）
├── 3-5 个精准标签
├── 1-2 个热门标签
└── 1 个长尾标签
```

### 封面图技巧

| 技巧 | 说明 |
|------|------|
| **色彩统一** | 选一个主色调，保持账号风格一致 |
| **大字标题** | 封面上加 6-8 个字的标题 |
| **产品居中** | 产品占画面 60% 以上 |
| **场景化** | 在使用场景中拍，不要白底 |
| **对比图** | 前后对比、大小对比最吸引人 |
